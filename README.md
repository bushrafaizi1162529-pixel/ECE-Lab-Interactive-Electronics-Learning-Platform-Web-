# ECE Lab Interactive Electronics Learning Platform

An interactive web-based platform for learning electronics and circuit design concepts.

## Getting Started

### Installation

1. Clone the repository:
```bash
git clone https://github.com/bushrafaizi1162529-pixel/ECE-Lab-Interactive-Electronics-Learning-Platform-Web-.git
cd ECE-Lab-Interactive-Electronics-Learning-Platform-Web-
```

2. Install dependencies:
```bash
npm install
```

### Development

To start the development server and run on localhost:

```bash
npm run dev
```

The application will be available at:
- **http://localhost:5173/**

To make the server accessible on your network, use:
```bash
npm run dev -- --host
```

### Build

To build for production:

```bash
npm run build
```
